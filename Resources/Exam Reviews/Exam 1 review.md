# Exam 1 Review, Spring 2017, CS 200

## Exam layout

* Paper exam
  * Worth 50% overall
  * 13 questions
  * Closed book, closed note
* Coding exam
  * Worth 50% overall
  * 3 problems
  * Open book, open note, can use class resources
  * No internet surfing

### Topics

* Paper exam
  * Variables and data types
  * Console input and output
  * Boolean expressions
  * Branching
  * Loops
  * Functions
  * Debugging
* Coding exam
  * Functions
  * If statements
  * Loops

Make sure you have your MPL questions done before the exam!
