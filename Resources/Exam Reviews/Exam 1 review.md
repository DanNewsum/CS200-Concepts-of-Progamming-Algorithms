# Exam 1 Review, Spring 2017, CS 200

* 50% - Paper exam - 13 questions
* 50% - Coding portion - 3 questions

### Topics

* Variables and data types
* Console input and output
* Boolean expressions
* Branching
* Loops
* Functions
* Debugging

Make sure you have your MPL questions done before the exam!
